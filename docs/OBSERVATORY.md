# Observatory

The **Observatory** tab consolidates live data from all enabled connectors into a single dashboard. The tab is hidden in the navigation bar until at least one connector is enabled in Parameters → Connectors.

---

## Navigation visibility

The Observatory nav item is hidden by default. It appears automatically when a connector is enabled and saved. Disabling all connectors hides it again.

---

## AllSky section

When the AllSky connector is enabled, the Observatory renders panels for each active module:

### Live image

Auto-refreshing all-sky camera image, reloaded every 30 seconds via the MyAstroBoard proxy. If the image fails to load (AllSky offline or path misconfigured), an "unavailable" placeholder is shown and a reload is retried after 5 minutes.

When sensor data is also enabled, a day/night badge is displayed alongside the image title (sourced from `DAY_OR_NIGHT` in the AllSky Export JSON).

### Sensor data

Live camera and environment readings polled every 60 seconds from the AllSky Export JSON. Requires the **AllSky Export** overlay module to be added to both Day and Night pipelines in AllSky settings.

Displayed fields (when available in the JSON):

| Label | AllSky key |
|-------|-----------|
| Temperature | `AS_TEMPERATURE_C` |
| Humidity | `AS_DEWCONTROLHUMIDITY` or `AS_HUMIDITY` |
| Dew point | `AS_DEWCONTROLDEW` |
| Dew heater | `AS_DEWCONTROLHEATER` |
| Gain | `AS_GAIN` |
| Exposure | `AS_sEXPOSURE` or `AS_EXPOSURE_US` |
| Mean brightness | `AS_MEAN` |
| AllSky version | `ALLSKY_VERSION` |

### Keogram

Daily keogram — a horizontal timeline strip showing sky evolution throughout the night. Generated by AllSky at end of night. Displays "not yet generated" until the file exists on the AllSky server.

### Startrails

Stacked startrails image for the current night. Generated by AllSky at end of night.

### Daily timelapse

Full-night timelapse video (MP4) generated by AllSky at end of night, embedded as an HTML5 `<video>` element.

### Mini-timelapse

Frequently updated short timelapse clip. Requires the mini-timelapse feature to be enabled in AllSky settings (Number Of Images > 0). Shown as a clickable thumbnail with a link to the full MP4 video.

---

## Layout

The panel grid adapts to which modules are active:

| Condition | Layout |
|-----------|--------|
| Live image only (no sensor) | Full-width image |
| Live image + sensor data | Image 8/12 cols, sensor panel 4/12 cols |
| Keogram alone | Full-width |
| Keogram + startrails or mini-timelapse | Keogram 9/12, neighbour 3/12 |
| Startrails / mini-timelapse | Half-width each |
